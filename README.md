# Blake Durand — Personal Website

A personal portfolio website for Blake Durand, showcasing real estate services, media production, and Airbnb property management.

## Tech Stack

- **React 19** + **TypeScript**
- **Vite** — build tool
- **Tailwind CSS v4** — styling
- **Framer Motion** — animations
- **React Router v7** — client-side routing
- **Lucide React** — icons

## Pages

- **Home** — Hero, services overview, and portfolio preview
- **Real Estate** — Direct booking listings, city search, and services for buyers/sellers/investors
- **Media** — Photography, cinematography, and post-production portfolio
- **Shop** — Digital products and consulting
- **Contact** — Contact form and info

## Getting Started

**Prerequisites:** Node.js 18+

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

Output will be in the `dist/` folder.
